# Skinlytics – Project Overview

## Project Goal

Skinlytics is a data analytics project I created to analyze skincare product data and turn it into useful business insights. Instead of spending hours going through hundreds of products one by one, this project uses data to quickly organize and analyze the information so businesses can make better decisions.

## Business Problem

Beauty retailers like Ulta sell thousands of skincare products from many different brands. Looking through all of that data manually would take a lot of time and make it difficult to spot trends. Skinlytics helps solve that problem by cleaning, organizing, and analyzing the data so it's easier to understand what customers actually like and what products perform the best.

## Business Questions

Some of the questions this project will answer include:

- Which skincare brands have the highest average ratings?
- Which ingredients appear most often in highly rated products?
- Which products have the most customer reviews?
- Is there a relationship between product price and customer ratings?
- Which skincare categories receive the highest customer satisfaction?

## Tools Used

- Python
- Pandas
- SQL
- Power BI
- Git & GitHub

## Expected Outcome

The goal of Skinlytics is to turn raw skincare data into clear and useful insights. By the end of the project, it will include cleaned datasets, data visualizations, an interactive dashboard, and recommendations that could help a company like Ulta better understand customer preferences and make smarter business decisions.
